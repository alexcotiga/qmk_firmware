# TOTEM split keyboard

TOTEM is 34 keys column-staggered split keyboard made by @geigeigeist. It uses the Seeed XIAO RP2040.

You can use this command to compile the firmware for vial
```
make totem:vial:uf2-split-left
make totem:vial:uf2-split-right
```
afterwards go into the .build folder and copy pasta the uf2 file and rename it for left/right
